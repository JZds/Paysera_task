# Paysera_task
Application for counting commission fees of cash in/out actions.

## Ready application
```
git clone https://github.com/JZds/Paysera_task.git
composer install
cd Paysera_task
```

## Run application
```
php script.php input.csv
```

## Run Phpunit Tests
```
php vendor/phpunit/phpunit/phpunit
```
